download COPSolver installation files

Download the desired module, go to the folder where the .zip file has been downloaded, unzip the file, then go to the folder of the unzipped file using the terminal and type $./run.sh

At the moment there are available .deb files for installation only for elementary OS.

Adaptation to other ubuntu -based platforms can be done by adjusting the copsolver.desktop file.

The code is available, so it is also possible to run the code using some software such as Code::Blocks or to create executable files for different platforms.

It is possible to install the complete software, or just the desired libraries.

It is also possible to install one library over the other. I will try to update the main.cpp file constantly in order to avoid possible conflicts.

More details about the installation are available in COPSolver manual.
